## مشارکت غیرفنی در فونت آراد
- بنظر می‌آید فونت آراد، در زبان های کوردی و اردو، حروف جایگزین شرطی، به طور کامل و دقیق پیاده سازی نشدند. باید فونت در این زبان ها آزمایش شود.
